# instagram-sdk OAuth 2.0

## Features

* Sign In with Instagram.
* Fetch Instagram user(id,FullName, profilepicture).

## Getting Started

Add this dependency to your app build.gradle file
```
dependencies {
	        implementation 'com.github.EsraaNayel:instagram-sdk:1.0.0'
	}
```
